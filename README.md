\# ArtSupplyStore - Магазин товаров для творчества



\## Описание

Веб-приложение для управления товарами для творчества. Реализовано на .NET 9 + Blazor Server + EF Core.



\## Технологии

\- .NET 9.0

\- Blazor Server

\- Entity Framework Core 9.0 + SQLite

\- Bootstrap 5

\- Docker



\## Быстрый запуск



\### Docker (рекомендуемый)

```bash

docker-compose up --build

\# Открыть http://localhost:8027
